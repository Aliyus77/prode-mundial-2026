ESTRUCTURA DE COMPANYCONFIG:
  {
    id: string;              // "default"
    name: string;            // "Mi Empresa PRODE"
    logo?: string;           // URL opcional (para futuro)
    updatedAt: Date;         // Timestamp de último cambio
    updatedBy: string;       // ID del usuario que modificó
  }

LOGICA DEL NAVBAR:
  const userDisplayName = user?.firstName && user?.lastName 
    ? `${user.firstName} ${user.lastName}`
    : user?.name || "Usuario";

  → Prioridad:
    1. firstName + lastName (si existen)
    2. name (si no existen firstName/lastName)
    3. "Usuario" (fallback)

PERSISTENCIA:
  localStorage["mockCompanyConfig"] = {
    id: "default",
    name: "Nombre Empresa",
    updatedAt: "2024-04-03T16:15:09.939Z",
    updatedBy: "1"
  }
