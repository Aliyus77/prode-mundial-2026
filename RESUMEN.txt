🐛 SOLUCIÓN DE PROBLEMAS
────────────────────────────────────────────────────────────────────

"Cannot find module":
  → npm install

"Port 5173 already in use":
  → npm run dev -- --port 5174

"node_modules corrupted":
  → rm -rf node_modules && npm install

"TypeScript errors":
  → npm run lint

"Blank page":
  → Clear browser cache (Ctrl+Shift+Del)
  → npm run dev (reiniciar)
