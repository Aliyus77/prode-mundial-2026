  ✓ La predicción se muestra en card verde
  ✓ No puedes cambiarla

Prueba 2: Bloqueo de 24 horas
  1. Abre DevTools (F12) → Console
  2. Ejecuta:
     const m = JSON.parse(localStorage.getItem('mockMatches') || '[]');
     m[0].scheduledDate = new Date(Date.now() + 12*3600000);
     localStorage.setItem('mockMatches', JSON.stringify(m));
     location.reload();
  3. Vuelve a "Grupos"
  4. El primer partido mostrará: "⚠️ 12 horas restantes"
  ✓ Inputs siguen habilitados

  1. Repite prueba 2, pero con: new Date(Date.now() - 3600000)
  2. El primer partido mostrará: "⏰ Límite pasado"
  ✓ Inputs estarán deshabilitados

💾 DATOS PERSISTENTES
════════════════════════════════════════════════════════════════════════════════

Todos los datos se guardan en localStorage:
  • mockUsers - Usuarios registrados
  • mockMatches - Partidos de la fase de grupos
  • mockPredictions - Predicciones de usuarios ⭐ NUEVO
  • mockPrizes - Premios creados
  • mockAwards - Premios entregados
  • mockBanner - Banner publicitario

Limpiar datos (si necesitas resetear):
  1. Abre DevTools (F12)
  2. Application → Local Storage
  3. Borra las claves que necesites o todas

O en Console ejecuta:
  localStorage.clear();
  location.reload();

🔧 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Problema: "npm run dev" no funciona
Solución:
  npm install
  npm run dev

Problema: Los archivos no aparecen
Solución:
  npm run setup
  npm run dev

Problema: localStorage lleno
Solución:
  localStorage.clear();
  location.reload();

Problema: Predicción no se guarda
Solución:
  • Verifica que estés logueado
  • Verifica que ambos campos tengan números
  • Abre DevTools para ver errores (F12 → Console)

Problema: Banner no aparece
Solución:
  • Debes cargar primero en Admin → Banner
  • Asegúrate de hacer clic "Publicar"

✨ NOTAS ESPECIALES
════════════════════════════════════════════════════════════════════════════════

1. GENERACIÓN AUTOMÁTICA
   El script generate-files.mjs se ejecuta automáticamente con "npm run dev"
   No necesitas hacer nada especial

2. MOCK DATA
   Todos los datos viven en localStorage. No hay backend real.
   Esto permite probar sin internet y es fácil de cambiar a API real

3. GRUPOS REALES
   Los 12 grupos y 48 equipos son del Mundo 2026 oficial
   Los partidos se generan automáticamente (6 por grupo)

4. BLOQUEO DE PREDICCIONES
   Usa la fecha actual del sistema. Si cambias la hora de tu PC,
   se recalcula automáticamente

5. PERSISTENCIA
   Todo se guarda en localStorage. Si borras datos del navegador,
   se pierden. Para desarrollo, usa localStorage.clear() en console
