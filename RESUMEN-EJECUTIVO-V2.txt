💡 DECISIONES TÉCNICAS
════════════════════════════════════════════════════════════════════════════════

STACK TECNOLÓGICO:
  • React 18+ (frontend)
  • TypeScript (tipado)
  • Vite (bundler)
  • Material-UI v5 (componentes)
  • Context API (estado)
  • localStorage (persistencia)

POR QUÉ ESTAS OPCIONES:
  ✓ React: Popular, comunidad grande, fácil de aprender
  ✓ TypeScript: Previene errores, código autoexplicativo
  ✓ Vite: Rápido, moderno, excelente dev experience
  ✓ Material-UI: Componentes profesionales, bien documentados
  ✓ Context: Simple pero poderoso, no necesitaba Redux
  ✓ localStorage: Perfecto para mock, fácil de cambiar a API
