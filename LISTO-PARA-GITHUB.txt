════════════════════════════════════════════════════════════════════════════════
🔑 GITHUB CREDENTIALS
════════════════════════════════════════════════════════════════════════════════

Necesitarás:
  • Cuenta en https://github.com/Aliyus77/
  • Personal Token O SSH Key

Crear Personal Token:
  1. https://github.com/settings/tokens
  2. "Generate new token (classic)"
  3. Nombre: PRODE-2026
  4. Selecciona ☑️ repo
  5. Generate y cópialo

Usar SSH Key (alternativa):
  1. ssh-keygen -t ed25519
  2. Copia ~/.ssh/id_ed25519.pub
  3. https://github.com/settings/keys → New SSH key
