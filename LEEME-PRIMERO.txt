📞 SOLUCIÓN DE PROBLEMAS RÁPIDA
════════════════════════════════════════════════════════════════════════

"Node no found"
→ Instalar desde https://nodejs.org

"npm install falla"
→ Limpiar: rm -rf node_modules && npm install

"Puerto 5173 en uso"
→ npm run dev -- --port 5174

"Blank page"
→ Limpiar cache (Ctrl+Shift+Del) y recargar

"Archivos src/ no existen"
→ node create-all-files.js

Ver consola del navegador (F12) para más detalles de errores.
