🔄 FLUJO DE ARCHIVOS DURANTE EJECUCIÓN
════════════════════════════════════════════════════════════════════════════════

1. Usuario ejecuta: npm run dev

2. package.json ejecuta:
   → "dev": "node generate-files.mjs && vite"

3. generate-files.mjs:
   ├── Crea directorios (src/, src/types/, etc.)
   ├── Genera todos los archivos .ts y .tsx
   ├── Initializa mock users en localStorage
   └── Termina

4. Vite inicia:
   ├── Compila TypeScript
   ├── Inicia servidor en http://localhost:5173
   └── Abre navegador

5. Usuario ve login page

6. Usuario puede:
   ├── Login (admin@prode.com/admin123)
   ├── Registrarse
   └── O ver demo con usuario existente
