PERSONAL TOKEN (Recomendado):
1. https://github.com/settings/tokens
2. Clic "Generate new token (classic)"
3. Nombre: PRODE-2026

SSH KEY (Alternativa):
1. ssh-keygen -t ed25519 -C "tu@email.com"
2. Presiona Enter 3 veces
3. Copia ~/.ssh/id_ed25519.pub
4. https://github.com/settings/keys → New SSH key
5. Pega tu clave pública
